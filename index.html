HERO - Different countries, different choices

The water-food nexus framework from the report's Figure 3.1, plotted with real data. 186 countries on freshwater stress (x) vs net food trade position (y), sized by population, colored by region. The 14 case study countries are highlighted, and the four contexts are labelled per the report.

About this prototype

Built with vanilla HTML + Observable Plot for the hero scatter. No build step. Static files only, hosted on GitHub Pages.

Data extraction lives in the parent project. See data/schema.md for column-by-column documentation including a sign-convention gotcha on the “Net Food Imports (%)” column.
